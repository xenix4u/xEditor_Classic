xEditor - Comments & Track Changes Demo

How to use:

  • Comments:
    • Select any text and click the Comment button in the toolbar
    • Click on highlighted text to view the comment thread
    • Reply to comments in the side panel
    • Edit or delete your own comments
    • Use Ctrl+Alt+M (or Cmd+Alt+M on Mac) to add a comment
  • Track Changes:
    • Click the Track Changes button to start tracking
    • All edits will be marked with author colors
    • View all changes in the changes panel
    • Accept or reject individual changes
    • Use Ctrl+Shift+E (or Cmd+Shift+E on Mac) to toggle tracking
